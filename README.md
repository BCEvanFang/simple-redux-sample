# A Simple Redux Example

[中文參考](https://cn.redux.js.org/)

[英文原文](https://redux.js.org/)

## Install & Build
```
npm install

npm run build
```

## 使用的庫
```
npm install --save redux
npm install --save react-redux
npm install --save-dev redux-devtools
```

## 執行
開啟 `index.html` ，從Console檢視輸出
